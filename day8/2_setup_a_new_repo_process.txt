   1. In local project / folder:
   git init (don't do it multiple times)


   2. git add .
   3. git commit -m"<commit-message>"
   4. git branch -M main

   5. Create a new remote (Github) repo. In this github repo, We can find command to connect remote to local repo.
      and copy that command

   6. Paste the remote command to local.
   7. git push origin main
